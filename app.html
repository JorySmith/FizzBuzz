FizzBuzz: A Coding Project

A program that loops through 1 to 100 checking to see if each number is a multiple or not of one or both of two user supplied numbers, for example "3" and "5", and displaying either the looped number, "Fizz", "Buzz", or "FizzBuzz" depending on which conditions are met.

If a looped number is a multiple of the first input number, "3", the program will display "Fizz". If the looped number is a multiple of "5", the program will display "Buzz". If the number is a multiple of "3" and "5", "FizzBuzz" will be displayed. If the number isn't a multiple of either "3" and "5", simply the number will be displayed.
